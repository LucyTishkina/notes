Proxies


Proxies
- sits between the users and the external network
- receives the user requests and sends the request on their behalf (the proxy)
- useful for caching information, access control, URL filtering, content scanning
- applications may need to know how to use the proxy (explicit)
- some proxies are invisible (transparent)

Application Proxies
- one of the simplest “proxies” is NAT
→ a network-level proxy
- most proxies in use are application proxies
→ the proxy understands the way application works
- a proxy may only know one application
→ HTTP
- many proxies are multipurpose proxies
→ HTTP, HTTPS, FTP, etc.

Forward proxy
- an “internal proxy”
→ commonly used to protect and control user access to the Internet

User -------------------- Proxy -------------- Internet
|----Internal Network------|

Reverse proxy
- inbound traffic from the Internet to your internal device

Internet --------------- Proxy -------------- Web Server
| -------- Internal network------|

Open proxy
- a third-party, uncontrolled proxy
→ can be a significant security concern
→ often used to circumvent existing security controls (like URL filters)
→ can modify traffic and malicious code

Internet --------------------- Proxy --------------------- Internet
