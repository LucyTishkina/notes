VPN Technologies


Site-to-Site VPNs
- Encrypt traffic between sites
→ through the public internet
- use existing Internet connection
→ no additional circuits or costs
- commonly done using VPN appliance
- traffic on the corporate network and remote site is in the clear, only traffic between the two sites is encrypted

Host-to-Site VPNs
- also called “remote access VPN”
- requires software on the user device
→ may be built-in to existing operating system
- using VPN concentrator

Host-to-host VPNs
- user to user encryption
- software-based
→ no hardware needed
