Symmetric Algorithms


DES
- Data Encryption Standard
→ DES and Triple DES
- developed between 1972 and 1977 by IBM and the NSA
→ one of the Federal Information Processing Standards (FIPS)
- 64-bit block cipher
→ 56-bit key (very small in modern terms)
- easily brute-forced with today's technology

3DES
- Triple DES
→ extends the use of the DES cipher
- 3 different keys is the strongest 3DES encryption method
→ 2 separate keys is deprecated, a single key isn't allowed
- use DES encryption/decryption 3 times
→ encrypt with 1ts key
→ decrypt with 2nd key
→ encrypt with 3rd key
- superseded by AES
→ Advanced Encryption Standard


Plaintext encrupted with DES key 1 => Ciphertext decrypted with DES key 2 => Ciphertext encrypted with DES key 3 => final ciphertext


RC4
- Rivest Cipher 4
→ Ron Rivest (Ron's Code 4)
- part of the ill-fated WEP standard
→ also part of SSL, but removed from TLS
- RC4 has “biased output”
→ if the 3rd byte of the original state is zero and the 2nd byte is not equalt to two then the 2nd output byte is always 0
- not common to see RC4 these days
→ WPA2 moved to AES

AES (Advanced Encryption Standard)
- US Federal Government Standard
→ FIPS 197 in 2001
→ it took 5 years to standardize this
→ developed by 2 Belgian cryptographers
⇒ Joan Daemen and Vincent Rijmen
- 128-bit block cipher
→ 128-, 192-, and 256-bit keys
- used in WPA2
→ powerful wireless encryption

Blowfish and twofish
- Blowfish
→ designed in 1993 by Bruce Schneier
→ 64-bit block cipher, variable length key (1-488 bits)
→ no known way to break the full 16 rounds of encryption
→ one of the first secure ciphers not limited by patents
- twofish
→ successor to Blowfish
→ 128-bit block size, key sizes up to 256
→ designed by Bruce Schneier, John Kelsey, Doug Whiting, David Wagner, Chris Hall, and Niels Ferguson, Stefan Lucks, Tadayoshi Kohno, Mike Stay
→ no patent, public domain
